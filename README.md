# Business Sales Performance Analytics

## Objective
Analyze business sales data to identify trends, top-performing products, categories, and regions.

## Dataset
Sample Superstore Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## KPIs
- Total Sales
- Total Profit
- Total Orders

## Analysis Performed
1. Monthly Sales Trend
2. Top Selling Products
3. Category Analysis
4. Region Analysis
5. Profit Analysis
6. Customer Segment Analysis

## Key Insights
- Identified highest performing products.
- Identified top revenue generating regions.
- Compared category-wise sales.
- Evaluated profitability across regions.

## Recommendations
- Focus on high-performing categories.
- Increase marketing in profitable regions.
- Optimize low-profit regions.
- Promote best-selling products.
